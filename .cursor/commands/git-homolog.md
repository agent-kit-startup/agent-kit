# Git homolog (legado)

Sinônimo legado de **`git staging`** — mesma rotina, ver [git-staging.md](git-staging.md).

1. Siga a seção "Prompt: git staging" em `autogit/gitupdate.md`.
2. Branch alvo: `staging` (ou `homologacao` em repositórios legados que ainda não migraram).
3. **Nunca** commit direto em `main`.
4. Ao concluir: atualize `.cursor/HANDOFF.md`; se couber, memory-loop WRITE.

Preferir `/git-staging` daqui em diante.
