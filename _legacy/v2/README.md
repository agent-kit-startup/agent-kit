# Agent Kit v2 (reference)

These files supported the drag-`install.md` flow and a JSON registry built from `.cursor/skills/`.

- **`install.md`** / **`adicionar-skills.md`** — v2 onboarding (superseded by v3 CLI).
- **`skills-registry.json`** / **`registry-schema.md`** / **`scripts/build-registry.sh`** — manual registry (v3 uses `registry/registry.json` and `agent-kit add`).
- **`plugin.json`** — Cursor plugin manifest v2 (v3: `.cursor-plugin/plugin.json`).
- **`HANDOFF.md.example`** — template copied by `cursor-handoff install`.
- **`categories.md`**, **`deslop.md`** — v2 notes (skills live under `.cursor/skills/` in this repo).
